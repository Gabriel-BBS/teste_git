HABILIDADES QUE APRENDI
- baixar atualizações do repositório remoto para o local - git pull ou git fetch + git merge
- git fetch é mais seguro por ele baixa mas não altera o repositorio local
- enviar atualizações do meu repositório local para o remoto - git push <remote name> <branch name>
(caso aconteça um erro, o arquivo do repositório remote não está igual ao da sua máquina local) --ATENÇÃO
- comando pull de um repo remoto para um local = git pull origin <remote branch name>
- deletar uma branch remota = git push -d origin <branch name>
- como rejeita ou aceitar alterações locais = VScode pergunta
